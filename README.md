## Tentang

## Instalasi

```sh
Version PHP 7.4.19
COMPOSER
```

Clone repository :

```sh
https://github.com/azmihaidar1/jayafutsal.git
```

Instalasi dependencies PHP dengan menggunakan Composer :

```sh
composer install
```

Buat key baru :

```sh
php artisan key:generate

Jalankan WEB :

```sh
php artisan serve

```
